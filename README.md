# mini-project-python-ml

关于机器学习的一些小项目

### **项目**



##### [预测学生成绩](backpropagation-students-grades)
[数据集](backpropagation-students-grades/binary.csv)
反向传播（Backpropagation）是深度学习的基础。TensorFlow 或者其它框架会替你把它做好，但是本项目完成了对神经网络的训练过程的实现。具体包括：
- 正向传播算法
- 反向传播算法
- 更新权重



###### **使用到的包**

```Numpy``` - 用于手动构建深度学习网络
```Pandas``` - 进行矩阵运算
